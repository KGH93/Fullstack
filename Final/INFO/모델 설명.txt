==============================

추천 모델로는 LightFM이라는 추천 시스템 라이브러리를 사용했습니다.
이 모델은 사용자와 아이템 간의 상호작용 데이터만을 기반으로 학습하는 협업 필터링(Collaborative Filtering) 방식입니다.

사용자가 쇼핑몰에서 실행하는 행동 —
예를 들어, 상품을 보기(view), 장바구니에 담기(cart), 구매하기(buy) — 는
모두 로그로 기록되며, 각 행동에는 **가중치(weight)**가 부여됩니다.

보기: 1	장바구니 담기: 3	구매하기: 5

전처리 단계에서는 이 가중치를 기반으로
사용자-아이템 간의 상호작용 행렬을 생성하고,
이 행렬을 LightFM 모델의 학습에 사용합니다.

모델은 이 행렬을 통해 학습을 진행하며,
특정 user_id가 주어지면 해당 사용자가 관심을 가질 확률이 높은 상품 6개를 추천합니다.

==================== 추가 설명 ====================

LightFM은 **상호작용 기반 협업 필터링(Pure CF)**과
사용자·상품의 부가 정보를 함께 사용하는 Hybrid 방식 모두를 지원하는 추천 모델입니다.
이 중 저희는 사용자와 상품 간의 상호작용 데이터만을 사용하는 Pure Collaborative Filtering 방식을 선택하였습니다.

user_id와 item_id로 구성된 **상호작용 행렬(interaction matrix)**과
행동에 따른 **가중치(weight)**만을 기반으로,
**사용자와 아이템의 latent factor(잠재 요인)**를 학습합니다.

Latent factor란, 데이터에 명시되지 않은 선호나 특성을 수치로 표현한 값으로,
LightFM은 이를 벡터 형태로 학습하고
사용자와 아이템 벡터의 **내적(dot product)**을 통해
해당 사용자가 아이템을 얼마나 선호할지를 예측합니다.
