크롬 확장프로그램으로 이미지 다운

--------윈도우 운영체제--------

WINDOW + E  - 파일탐색기
WINDOW + D  - 바탕화면 바로가기
WINDOW + TAB  - 작업화면 전환
WINDOW + 방향키  - 작업화면 나누기


-------VSCODE 단축기-------

! + TAB : HTML 기본 템플릿 생성

CTRL + / : 주석

ALT + SHIFT + 방향키 : 줄복사 붙이기

CTRL + D : 다중선택 , 선택한 텍스트와 동일한 다른 텍스트를 한개씩 추가해 수정

CTRL + SHIFT + L : 다중선택, 선택한 텍스트와 동일한 다른 텍스트를 한번에 추가해 수정

CTRL + SHIFT + K : 한줄지우기
ALT + SHIFT + F : 코드 포멧팅 , 줄 정리 

CTRL + \ : 두개 이상의 파일을 동시에 편집

ALT + B : HTML 창띄우기

ALT + L + O : HTML 라이브서버로 창띄우기
